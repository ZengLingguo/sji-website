Recommended Inquiry Information

  • Code type: 1D barcode, QR code, Data Matrix, DPM, or mixed code types.
  • Reading mode: fixed installation, compact embedded installation, handheld scanning, PDA, or software decoding.
  • Working distance, code size, field of view, line speed, interface requirements, and expected quantity.
  • Sample code images and application photos help SJI recommend a more suitable product family.
